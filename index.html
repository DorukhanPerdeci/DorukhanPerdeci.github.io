Bootstrap Accordion Yapısı

Bu yazımızda bootstrap kullanarak accordion yapısına bakacağız.

Öncelikle bootstrap kullanabilmek için bize gerekli olan dosyalar bootstrap sitesinden indirmemiz gerekiyor. Bkz:(dosyaları indirmek için tıklayın.)

İndireceğimiz dosyalar Bootstrap'in "CSS" ve "JS" dosyalarıdır. Herhangi bir html dosyası oluştururken html kodlarımız arasına bu indirdiğimiz dosyaları çağırmamız gerekir ki bootstrap'in kütüphanelerini kullanabilelim.

Dilerseniz hiç dosyalarla uğraşmak istemiyorsanız buraya tıklayarak gideceğiniz kısımdan kodunuza "CSS" ve "JS" kodlarını bootstrap'in sunucularından çağırarak da yapabilirsiniz fakat css kodlarını düzenlemeniz zorlaşabilir. Örnek siteniz için başlangıç kodu aşağıdadır. 

                        
<!doctype html>
<html lang="en">
  <head>
    <!-- Required meta tags -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">

    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous">

    <title>Hello, world!</title>
  </head>
  <body>
    <h1>Hello, world!</h1>

    <!-- Optional JavaScript; choose one of the two! -->

    <!-- Option 1: Bootstrap Bundle with Popper -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-ka7Sk0Gln4gmtz2MlQnikT1wXgYsOg+OMhuP+IlRH9sENBO0LRn5q+8nbTov4+1p" crossorigin="anonymous"></script>

    <!-- Option 2: Separate Popper and Bootstrap JS -->
    <!--
    <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.10.2/dist/umd/popper.min.js" integrity="sha384-7+zCNj/IqJ95wo16oMtfsKbZ9ccEh31eOz1HGyDuCQ6wgnyJNSYdrPa03rtR1zdB" crossorigin="anonymous"></script>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.min.js" integrity="sha384-QJHtvGhmr9XOIpI6YVutG+2QOK9T+ZnN4kzFN1RtK3zEFEIsxhlmWl5/YESvpZ13" crossorigin="anonymous"></script>

  </body>
</html>

Kodumuzu hazırladığımza göre bootstrap ile accordion yapısına geçebiliriz.

Bootstrap'in kendi sitesinden "GET BOOTSTRAP" diyerek girdiğimiz sayfada bulunan sol tarafta bulunan "Components" kategorisinin altındaki Accordion yapısına girelim. Bkz:(ilgili bölüme gitmek için tıklayın.

Bu kısımda bize gösterilen yerlerde accordion yapımızın nasıl çalışılmasını istediğimze göre farklı türlerde accordion yapılarını bootstrap karşımıza getiriyor.

Yukarıda sizlere bootstrap ile accordion yapısını gösterdim. Gerekli alandaki kodları bootstrap html kodlarınızda uygun bir yere yerleştirebilirsiniz.

Eğer ki kodunuz ile ilgili bir hata ya da görsel problem yaşarsanız öncelikle kullandığınız bootstrap sürümünüzü ve kodları aldığınız bootstrap sürümünü kontrol ediniz.
